Key Features

Source-First Analysis

4-tier source hierarchy: issuer filings → regulators → official datasets → secondary. Global market routers for US, Canada, UK, Australia, Japan, Korea, HK, China, and more.

Source Quality

Gate-Driven Valuation

3 mandatory gating tables block weak evidence. Valuation is blocked when capital structure is unverified, commercial evidence is weak, or regulatory gates are unresolved.

Valuation Gate

Bloomberg Terminal Integration

Agent assesses data confidence after online search. When data is insufficient, conflicting, or stale, it generates structured Bloomberg Terminal requests with function references.

v1.3

Financial Analysis Framework

Revenue quality decomposition (pass-through vs value-creating), earnings quality, cash flow decomposition, ROIC/ROE, and variance analysis with materiality thresholds.

v1.2

Semantic Claim Controls

Gates for moat, platform, data flywheel, network effect, and high margin business. Claims require minimum evidence or are downgraded to fallback labels.

Financial Analysis Framework

A. Revenue Quality

Decomposes revenue into value-creating vs pass-through. Calculates core margin after stripping procurement, hardware resale, and sub-contractor pass-through.

B. Earnings Quality

GAAP vs non-GAAP bridge, SBC impact, accrual ratio, ASC 606 revenue recognition checks.

C. Cash Flow

Maintenance vs growth capex, FCF variants, working capital decomposition (DSO, DIO, DPO).

D. Return & Leverage

ROIC, ROE DuPont decomposition, tangible ROIC, net debt/EBITDA, interest coverage.

E. Variance Analysis

Price/volume/mix decomposition, materiality thresholds (15% YoY, 5% vs guidance), waterfall format.
