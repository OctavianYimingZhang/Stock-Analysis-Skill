# Angle Library

Use this file to choose the right analysis angle for a stock. Treat every entry as a framing aid only. Re-source all facts from authoritative sources at runtime.

## Recurring Angle Clusters

- AI infrastructure bottlenecks and capex beneficiaries

- Platform, software, fintech, and healthcare flywheels
- Turnaround, debt repair, and operating leverage
- Resource scarcity and industry-chain positioning
- Commercialization, certification, and milestone gating
